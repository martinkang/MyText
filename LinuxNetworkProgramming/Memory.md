# Memory

## 메모리 영역의 종류
* text
	- 일반적으로 프로그램의 실행 코드가 존재하는 영역.
	- 특수한 경우에 데이터 영역에 코드를 포함하여 컴파일 가능 
		- 보통 보안 관련 프로그래밍

* data
	- static 으로 사용되는 심벌을 가진 영역
	* 쓰임새에 따라서 3가지로 나뉜다.
		- 읽기 전용으로 초기화 되는 영역
			* .rodata 영역
			* const 로 선언되는 영역
			* 그 외에 시스템에서 사용된 각종 문자역 등이 이곳에 위치.
				- printf 문의 포맷 스트링이나 상수 문자열 등
		- 읽기/쓰기가 가능한 영역으로 초기화 되는 영역
		- 초기화되지 않는 영역
		

* stack


* heap
