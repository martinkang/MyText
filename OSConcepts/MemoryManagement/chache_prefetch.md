# Cache 와 Prefetch

## Cache
- 캐싱은 값싸고 느린 저장 장치의 데이터 중 자주 사용하는 일부를 비싸고 빠른 저장 장치치로 옮겨서 사용하는 기법.


```c++
int main()
{
	int a[20] = { 10, 20, 10, 45, 4, 1, 5, 2, 3, 1, 2, 4, 5, 123, 4, 6, 70, 54, 64, 30 };
	int b[20], c[20], d[20], e[20];

	int i = 0;

	for ( i = 0; i < 20; i++ )
	{
		b[i] = a[i];
		c[i] = a[i];
	}

	for ( i = 0; i < 20; i++ )
	{
		d[i] = a[i] + i;
		e[i] = a[i] * 2;
	}

	return 0;
}
```
